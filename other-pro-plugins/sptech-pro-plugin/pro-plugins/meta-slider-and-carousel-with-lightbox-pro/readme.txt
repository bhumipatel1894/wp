Plugin add a gallery meta box in your post, page and create a Image gallery menu tab. Display with a lightbox. 

== Description ==
A very simple plugin to add gallery meta box in your post, page and and create a Image gallery menu tab in WordPress menu section and display Gallery on frontend of your website in a slider OR carousel view with the help of shorcode. The gallery field provides a simple and intuitive interface for managing a collection of images.

View [DEMO](http://wponlinesupport.com/wp-plugin/meta-slider-carousel-lightbox/)  for more details.


= This plugin contain Two shortcode: =
1) Slider View
<code>[meta_gallery_slider]</code>

2) Carousel View
<code>[meta_gallery_carousel]</code>

2) Variable Width View
<code>[meta_gallery_variable]</code>

= Gallery Slider Shortcode =
<code>[meta_gallery_slider show_title="true" show_caption="true" show_description="true" slider_height="500" design="design-1" dots="true" arrows="true" autoplay="true" autoplay_speed="3000" speed="500" loop="true"]</code>

= Gallery Carousel Slider Shortcode =
<code>[meta_gallery_carousel design="design-1" show_title="true" show_caption="true" show_description="true" slider_height="500" slide_to_show="2" slide_to_scroll="2" dots="true" arrows="true" autoplay="true" autoplay_speed="3000" speed="500" centermode="false" loop="true"]</code>

= Gallery Carousel Slider Shortcode =
<code>[meta_gallery_variable show_title="true" show_caption="true" show_description="true" slider_height="500" design="design-1" dots="true" arrows="true" autoplay="true" autoplay_speed="3000" speed="500" loop="true"]</code>

= You can use Following parameters with shortcode =
= Gallery Slider =
<code>[meta_gallery_slider]</code>
* **Id:** [meta_gallery_slider id="5"] (Enter gallery post id to display it's gallery. Leave empty for default post id.)
* **Show Title:** [meta_gallery_slider show_title="true"] (Hide/Show image title. By default value is "true". Values are "true" OR "false")
* **Show Caption:** [meta_gallery_slider show_caption="true"] (Hide/show image caption. By default value is "true". Values are "true" OR "false")
* **Show Description:** [meta_gallery_slider show_description="true"] (Hide/show image caption. By default value is "true". Values are "true" OR "false")
* **Slider Height:** [meta_gallery_slider slider_height="500"] (Control slider height.)
* **Design:** [meta_gallery_slider design="design-1"] (Control slider design. Values are design-1 to design-15.)
* **Pagination and Arrows:** [meta_gallery_slider dots="true" arrows="true"] (Hide/Show pagination and arrows. By default value is "true". Values are "true" OR "false")
* **Autoplay and Autoplay Speed:** [meta_gallery_slider autoplay="true" autoplay_speed="3000"]
* **Slide Speed:** [meta_gallery_slider speed="3000"] (Control the speed of the slider.)
* **Loop:** [meta_gallery_slider loop="true"] (Run slider contineously.)
* **Navigation:**  [meta_gallery_slider navigation="true"] (Enable slider navigation. Values are "true" or "false". By default value is false.)
* **Navigation columns:** [meta_gallery_slider nav_slide_column="5"] (Number of image columns you want to show in navigation.)
* **Navigation image size:** [meta_gallery_slider nav_image_size="medium"] (Values are 'thumbnail', 'medium', 'large', 'full'.)
* **Popup:** [meta_gallery_slider popup="true"] (Enable lightbox popup. Values are "true" OR "false")

= Gallery Carousel =
<code>[meta_gallery_carousel]</code>
* **Id:** [meta_gallery_carousel id="5"] (Enter gallery post id to display it's gallery. Leave empty for default post id.)
* **Design:** [meta_gallery_carousel design="design-1"] (Control slider design. Values are design-1 to design-15.)
* **Show Title:** [meta_gallery_carousel show_title="true"] (Hide/Show image title. By default value is "true". Values are "true" OR "false" )
* **Show Caption:** [meta_gallery_carousel show_caption="true"] (Hide/show image caption. By default value is "true". Values are "true" OR "false")
* **Show Description:** [meta_gallery_carousel show_description="true"] (Hide/show image caption. By default value is "true". Values are "true" OR "false")
* **Slider Height:** [meta_gallery_carousel slider_height="500"] (Control slider height.)
* **Display number of images at time:** [meta_gallery_carousel slide_to_show="2"] (Display no of image in a slider.)
* **Number of images slides at a time:** [meta_gallery_carousel slide_to_scroll="2"] (Controls number of images slide at a time)
* **Pagination and Arrows:** [meta_gallery_carousel dots="true" arrows="true"] (Hide/Show pagination and arrows. By default value is "true". Values are "true" OR "false")
* **Autoplay and Autoplay Speed:** [meta_gallery_carousel autoplay="true" autoplay_speed="3000"]
* **Slide Speed:** [meta_gallery_carousel speed="3000"] (Control the speed of the slider.)
* **Slider Center Mode:** [meta_gallery_carousel centermode="false"] (Enable slider center mode effect.)
* **Loop:** [meta_gallery_carousel loop="true"] (Run slider contineously.)
* **Popup:** [meta_gallery_carousel popup="true"] (Enable lightbox popup. Values are "true" OR "false")

= Variable Gallery =
<code>[meta_gallery_variable]</code>
* **Id:** [meta_gallery_variable id="5"] (Enter gallery post id to display it's gallery. Leave empty for default post id.)
* **Design:** [meta_gallery_variable design="design-1"] (Control slider design. Values are design-1 to design-15.)
* **Show Title:** [meta_gallery_variable show_title="true"] (Hide/Show image title. By default value is "true". Values are "true" OR "false")
* **Show Caption:** [meta_gallery_variable show_caption="true"] (Hide/show image caption. By default value is "true". Values are "true" OR "false")
* **Show Description:** [meta_gallery_variable show_description="true"] (Hide/show image caption. By default value is "true". Values are "true" OR "false")
* **Slider Height:** [meta_gallery_variable slider_height="500"] (Control slider height.)
* **Pagination and Arrows:** [meta_gallery_variable dots="false" arrows="false"] (Hide/Show pagination and arrows. By default value is "true". Values are "true" OR "false")
* **Autoplay and Autoplay Speed:** [meta_gallery_variable autoplay="true" autoplay_speed="1000"]
* **Slide Speed:** [meta_gallery_variable speed="3000"] (Control the speed of the slider.)
* **Loop:** [meta_gallery_variable loop="true"] (Run slider contineously.)
* **Navigation:**  [meta_gallery_variable navigation="true"] (Enable slider navigation. Values are "true" or "false". By default value is false.)
* **Navigation columns:** [meta_gallery_variable nav_slide_column="5"] (Number of image columns you want to show in navigation.)
* **Navigation image size:** [meta_gallery_variable nav_image_size="medium"] (Values are 'thumbnail', 'medium', 'large', 'full'.)
* **Popup:** [meta_gallery_variable popup="true"] (Enable lightbox popup. Values are "true" OR "false")


== Installation ==
1. Upload the 'meta-slider-and-carousel-with-lightbox-pro' folder to the '/wp-content/plugins/' directory.
2. Activate the "meta-slider-and-carousel-with-lightbox-pro" list plugin through the 'Plugins' menu in WordPress.

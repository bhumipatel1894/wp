== Description ==
WooCommerce Product Slider/Carousel and Grid is a best product slider to display your WooCommerce Product, Best Selling Product and Featured Product in Slider or Grid view with 15 cool designs. You can easily display this product slider in your theme using shortcode.

You can short product by category by adding category ID in the shortcode as a shortcode parameter.

WooCommerce product slider / carousel allows you to showcase your products in a nice sliding manner.

This plugin using the original loop form WooCommerce thats means it will display your product design from your theme plus style.

View [DEMO](http://wponlinesupport.com/wp-plugin/woo-product-slider-carousel-category/) for more details.

= This plugin contains 3 shortcodes: =

1) Display WooCommerce **product in slider / carousel** view
<code>[wcpscwc_pdt_slider] OR [wcpscwc_pdt_slider cats="CATEGORY-ID"]</code>

2) Display WooCommerce **Best Selling Product in slider / carousel view**
<code>[wcpscwc_bestselling_pdt_slider] OR [wcpscwc_bestselling_pdt_slider cats="CATEGORY-ID"]</code>

3) Display WooCommerce **Featured Product in slider / carousel view**
<code>[wcpscwc_feat_pdt_slider] OR [wcpscwc_feat_pdt_slider cats="CATEGORY-ID"]</code>

= You can use Following parameters with all 3 shortcode =

* **Design:** design="design-1" (Added a design parameter for custom designing. By default it will take design from wooCommerce OR from your theme. Values are design-1 to design-15.)
* **Display Product by category:** cats="category-ID"  (Show your product slider by category. values are comma separated category id. by default it shows all category products.)
* **Don't Display Category** exclude_cat="32,19,42" (Exclude some category from your product slider.You can enter multiple category id with comma seperated.)
NOTE: When "cats" parameter used "exclude_cat" parameter will be ignored.
* **Limit:** limit="5" ( ie Display 5 product at time. By defoult value is -1 ie all.)
* **Display number of products at time:** slide_to_show="2" (Display no of products in a slider.)
* **Number of products slides at a time:** slide_to_scroll="2" (Controls number of products rotate at a time.)
* **Pagination and arrows:** dots="false" arrows="false" (Hide/Show pagination and arrows. By defoult value is "true". Values are true OR false)
* **Autoplay:** autoplay="true" (Start slider automatically. Values are "true" OR "false".)
* **Autoplay Speed** autoplay_speed="1000" (Delay between two slides.)
* **Loop:** loop="true" (Run slider contineously. Values are true OR false)
* **Slide Speed:** speed="3000" (Control the speed of the slider)
* **Display Specific Post (Posts):** posts="1,5,8" (Display only specific Products. You can add multiple post ids with comma seperated.)
* **Exclude Post:** exclude_post="4,7,9" (You cam exclude some Products which you do not want to display. You can add multiple post ids with comma seperated.)
* **Show Category:** show_category="true" (Want to show Product category name or not ? Values are true OR false.)
* **Product Ratings:** show_rating="true" (Want to show Product ratings or not ? Values are true OR false.)
* **Order Products (Orderby):** orderby="post_date" (Display Products in your desired order. Values are "post_date", "modified", "title", "name" (Post Slug), "ID", "rand", "menu_order".)
NOTE: orderby will not work with "wcpscwc_bestselling_pdt_slider" Shortcode.
* **Sort By (Order):** order="DESC" (Display Products in an 'Ascending' or 'Descending' order.)
NOTE: order will not work with "wcpscwc_bestselling_pdt_slider" Shortcode.
* **Product Sale Tag:** show_sale="true" (Want to show Product Sale Tag or not ? Values are true OR false.)
* **Link Behavior:** link_target="self" (Want to show Product in same tab or new tab ? Values are self OR blank.)
* **Product Description:** show_desc="true" (show Product description or not ? Values are true OR false.)
* **Products Grid View:** type="grid" (Show your products in grid view. by default value is slider. values are slider OR grid.)
* ** Grid Number:** grid="2" (Number of product you want to show per row.)
* **Adaptive Height** adaptiveheight="true" (Adjust your slider height accroding to your products image or conetnt. by default value is false. values are true OR false.)
* **Center Mode** centermode="true" (Slider center mode effect. by default value is false. values are true OR false.)

2. Activate the "woocommerce-product-slider-and-carousel-with-category" list plugin through the 'Plugins' menu in WordPress.
